## 方法
随机森林Random Forest


## 分层随机采样
80%的训练集和20%的测试集训练模型，采用训练集fit,测试集transform，防止测试集信息泄露到训练过程中


## 特征分类
数据特征采用了z-score标准化，保证模型不会因为特征数值大小不同造成偏差
（举个例子哈，Ageing Temp=500，Si=0.125, 前者权重太大）
类别特征这种采用One-Hot编码，简单的说就是把类别信息换成可以理解的数字


## 预测
对UTS，YTS和EL进行预测
